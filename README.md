# Locally-Refined-Graph-Neural-Network-(LRGNN)-data-and-codes

LRGNN is a graph neural network-based surrogate model designed specifically for irregular (non-uniform/unstructured) meshes.  
It serves as an efficient approximation surrogate to numerical simulations on complex geometries where regular grid-based surrogate methods are difficult to apply.
